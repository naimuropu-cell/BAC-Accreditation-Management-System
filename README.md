# 📚 BAC Accreditation Management System

A web-based Accreditation Evidence & Documentation Management System developed for **Green University of Bangladesh** to simplify and centralize the evidence collection process required for **Bangladesh Accreditation Council (BAC)** accreditation.

---

## 📖 Overview

The BAC Accreditation Management System is designed to replace scattered Google Drive folders, Excel sheets, and manual documentation with a centralized web application.

The system enables departments to organize accreditation evidence according to BAC standards, making document management faster, more secure, and easier during accreditation preparation.

---

## ✨ Features

- 📂 Evidence & document management
- 📑 BAC Standard-wise organization (Standard 1 – Standard 10)
- 🔍 Search and filter documents
- 👥 User Authentication (JWT)
- 🔐 Role-based access control
- 📎 Evidence link management
- 📊 Document status tracking
- 📁 Department-wise document organization
- 📱 Responsive UI
- ⚡ Fast and secure document retrieval

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/naimuropu-cell/BAC-Accreditation-Management-System.git
```

Go to project directory

```bash
cd BAC-Accreditation-Management-System
```

Install frontend dependencies

```bash
cd client
npm install
```

Install backend dependencies

```bash
cd ../server
npm install
```

Run backend

```bash
npm start
```

Run frontend

```bash
npm start
```
